# Arduino Door Alarm System

This project is a basic door alarm system developed using Arduino.

## Features
- Door status detection using magnetic reed switch
- Password-based control using keypad
- Alarm activation with passive buzzer
- Incorrect password keeps alarm active

## Components Used
- Arduino UNO
- Keypad
- Breadboard
- Magnetic reed switch
- Passive buzzer

## Pin Connections
- Keypad → D2–D8
- Door sensor → D11 & GND
- Buzzer → D10 & GND

## Software
- Arduino IDE
- Keypad.h library (must be installed)
